# CSS Style sheet
-----
## css 작성위치
* 내부스타일시트 <head><style>여기작성</style></head>`
* 외부스타일시트 style /파일명. css별도 생성 후 
    <link rel="stylesheet" href="./styles/파일명.css>`
    ----------
## font-family 글꼴걸정
* 선택자 {font-family:'대표글꼴','보조글꼴'}
* 모든 사용자에게 동일한 글꼴을 보여줄 수 있도록 웹 글꼴을 사용하는 것이 좋다!
